2. **Systems** - Operating systems, networks, distributed computing
3. **Software** - Software engineering, programming languages
4. **AI** - Machine learning, NLP, computer vision

### Qualifying Event (Not a traditional exam!)
- Take 3 courses covering 3 different pillars
- Must get **A grade** in all 3 courses
- Typically done in first 2 years

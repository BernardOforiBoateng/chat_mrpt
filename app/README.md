# ChatMRPT Application

Main Flask application for ChatMRPT - AI-powered malaria risk analysis platform.

## Structure

- **analysis/** - Risk analysis pipelines and algorithms
- **auth/** - User authentication system
- **config/** - Environment configurations
- **core/** - Core utilities, LLM management, request interpretation
- **data/** - Data processing and unified dataset construction
- **data_analysis_v3/** - LangGraph-powered data analysis
- **helpers/** - User assistance utilities
- **interaction/** - Interaction tracking
- **models/** - Data models
- **services/** - Business logic and services
- **static/** - Static assets (JS, CSS, visualizations)
- **survey/** - Survey and assessment system
- **templates/** - Jinja2 HTML templates
- **tools/** - Analysis tools
- **tpr_module/** - TPR analysis module
- **utils/** - Shared utilities
- **web/** - Flask blueprints and routes

## Purpose

Implements a conversational AI platform for malaria risk assessment, intervention planning, and epidemiological analysis with geospatial data integration.

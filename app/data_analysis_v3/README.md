# Data Analysis V3

Advanced data analysis system with LangGraph integration for conversational data exploration.

## Structure

- **core/** - Agent, state manager, TPR workflow handler, execution engine
- **formatters/** - Response formatting utilities
- **prompts/** - System prompts and templates
- **tools/** - Python execution tool, TPR analysis tool
- **tpr/** - TPR-specific workflow components

## Purpose

Provides LangGraph-powered conversational data analysis with Python code execution, TPR workflow orchestration, and intelligent response formatting. Enables users to explore data through natural language queries.

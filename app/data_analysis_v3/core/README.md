# Data Analysis V3 Core

Core components for LangGraph-powered data analysis.

## Key Files

- **agent.py** - Main LangGraph agent orchestrating analysis workflow
- **state_manager.py** - Analysis session state management
- **tpr_workflow_handler.py** - TPR-specific workflow coordination
- **executor.py** - Python code execution engine
- **metadata_cache.py** - Dataset metadata caching

## Purpose

Implements the agent-based conversational data analysis system using LangGraph for stateful workflows, dynamic tool execution, and intelligent response generation.

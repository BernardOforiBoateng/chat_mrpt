# Analysis Tools

Modular analysis tools for malaria risk assessment and intervention planning.

## Key Tools

- **complete_analysis_tools.py** - End-to-end analysis workflows
- **visualization_maps_tools.py** - Interactive map generation (composite, PCA, ITN)
- **visualization_charts_tools.py** - Chart and graph generation
- **itn_planning_tools.py** - ITN distribution planning and optimization
- **export_tools.py** - Data and report export utilities
- **variable_distribution.py** - Variable distribution analysis

## Purpose

Provides self-contained, callable tools for specific analysis tasks. Each tool follows a standard interface and can be invoked directly by the LLM request interpreter or through conversational commands.

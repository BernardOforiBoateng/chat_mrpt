# Visualization Charts

Chart generation utilities for data visualization.

## Components

- **distribution.py** - Distribution plots and histograms
- **relationship.py** - Scatter plots and correlation matrices
- **ranking.py** - Bar charts and ranking visualizations
- **comparative.py** - Comparison charts
- **spatial.py** - Spatial data visualizations
- **common.py** - Shared chart utilities

## Purpose

Generates interactive charts using Plotly for various data visualization needs in malaria risk analysis.

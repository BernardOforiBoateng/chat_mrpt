# Chat Handlers

Modular chat request handlers for sync and streaming conversations.

## Key Files

- **handlers.py** - Core sync and streaming chat logic
- **arena.py** - Arena battle and voting system
- **routing.py** - Mistral routing and chunk streaming helpers

## Purpose

Handles chat requests, manages streaming responses, coordinates arena battles, and routes between different LLM providers. Extracted from monolithic routes for better testability.

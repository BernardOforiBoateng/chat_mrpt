# Data Module

Data processing, loading, and unified dataset construction.

## Key Components

- **unified_dataset/** - Modular unified dataset builder (refactored from 2,167 lines to 704)
- **population_data/** - Population-based intervention (PBI) data loaders
- **processing.py** - Data validation and cleaning utilities

## Purpose

Handles data ingestion from CSV/shapefiles, constructs unified datasets combining demographic and geospatial data, and manages population distribution data for ITN planning.

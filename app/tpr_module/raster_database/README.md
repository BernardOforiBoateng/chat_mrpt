# Raster Database

Organized geospatial raster data for TPR analysis.

## Categories

- **elevation/** - Static elevation data
- **flood/** - Flood extent rasters (2018, 2021)
- **health/** - Malaria prevalence (PfPR) data (2020-2022)
- **housing/** - Housing quality (2015)
- **rainfall/** - Monthly rainfall data
- **soil/** - Soil wetness/moisture data
- **temperature/** - Monthly temperature data
- **urban/** - Nighttime lights and urbanization
- **vegetation/** - EVI, NDMI, NDWI vegetation indices
- **water_bodies/** - Distance to water bodies

## Purpose

Stores environmental and geospatial rasters with associated metadata for extracting ward-level statistics in TPR analysis.

# TPR Core

Core TPR analysis components and workflow orchestration.

## Key Files

- **tpr_calculator.py** - TPR calculation algorithms
- **tpr_pipeline.py** - Analysis pipeline execution
- **tpr_conversation_manager.py** - Conversational TPR workflow
- **tpr_state_manager.py** - TPR session state management

## Purpose

Implements the core logic for Test Positivity Rate calculations, pipeline execution, and conversational workflow management.

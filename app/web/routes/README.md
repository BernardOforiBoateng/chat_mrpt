# Web Routes

Flask blueprint routes for HTTP endpoints.

## Key Blueprints

- **analysis/** - Analysis chat routes
- **core_routes.py** - Main application routes (home, health checks)
- **upload_routes.py** - File upload endpoints
- **visualization_routes.py** - Visualization serving routes
- **export_routes.py** - Data export endpoints
- **arena_routes.py** - LLM arena battle routes
- **data_analysis_v3_routes.py** - V3 data analysis endpoints

## Purpose

Organizes HTTP routes into logical blueprints, each handling specific areas of functionality for the web application.

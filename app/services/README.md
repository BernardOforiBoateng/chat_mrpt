# Services Module

Service layer providing specialized agents and business logic.

## Key Components

- **agents/visualizations/** - Visualization generation agents (composite, PCA, TPR)
- **reports/** - Modern report generation system
- **container.py** - Dependency injection service container
- **conversational_data_access.py** - Natural language data querying
- **universal_viz_explainer.py** - AI-powered visualization explanations

## Purpose

Encapsulates business logic and provides reusable services for report generation, data access, visualization, and AI-powered analysis explanations.

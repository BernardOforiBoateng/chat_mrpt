# Chat Components

Conversational interface components for user interaction.

## Components

- **ChatContainer.tsx** - Main chat container
- **MessageList.tsx** - Scrollable message history
- **InputArea.tsx** - User input field and controls
- **RegularMessage.tsx** - Standard chat message
- **ArenaMessage.tsx** - Arena battle message with dual responses
- **SystemMessage.tsx** - System notifications
- **ClarificationMessage.tsx** - Clarification prompts

## Purpose

Implements the conversational UI with support for regular chat, streaming responses, arena battles, and system messages.

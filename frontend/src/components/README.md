# React Components

UI components organized by feature area.

## Structure

- **Chat/** - Chat interface components (messages, input, streaming)
- **Analysis/** - Analysis display components (dual method, method cards)
- **Arena/** - Arena mode components (dual response, voting)
- **Visualization/** - Visualization rendering and controls
- **Modal/** - Modal dialogs (upload, settings, education)
- **Auth/** - Authentication components (login, signup)
- **Toolbar/** - Top toolbar and controls
- **Sidebar/** - Navigation sidebar
- **Export/** - Export functionality
- **Profile/** - User profile components
- **Shared/** - Shared/reusable components

## Purpose

Each subfolder contains components for a specific feature area, promoting modularity and code organization.

# Modal Components

Modal dialog components for various interactions.

## Components

- **Modal.tsx** - Base modal component
- **UploadModal.tsx** - File upload dialog
- **SettingsModal.tsx** - Application settings
- **EducationPanel.tsx** - Educational content
- **ConfirmationModal.tsx** - Confirmation dialogs
- **QuickStartModal.tsx** - Onboarding guide

## Purpose

Provides modal dialogs for file uploads, settings, education, confirmations, and user onboarding.

# Visualization Components

Visualization rendering and interaction components.

## Components

- **VisualizationContainer.tsx** - Main visualization container
- **VisualizationFrame.tsx** - iframe-based HTML visualization rendering
- **VisualizationControls.tsx** - Visualization interaction controls

## Purpose

Renders interactive visualizations (maps, charts) generated by the backend and provides controls for fullscreen, export, and other interactions.
